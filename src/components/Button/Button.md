Default Button:

```js
<Button text='button' />
```

Primary Button:

```js
<Button variant='primary' text='button'  />
<Button variant='primary' text='button' rounded />
```

Secondary Button:

```js
<Button variant='secondary' text='button' />
```

Warning Button:

```js
<Button state='warning' text='warning button' />
```

Alert Button:

```js
<Button state='alert' text='alert button' />
```
